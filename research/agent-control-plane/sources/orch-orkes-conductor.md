# Orkes Conductor (orkes.io / conductor-oss.org)

- **Accessed:** 2026-07-13
- **Type:** Primary (orkes.io) + labeled press
- **Confidence:** high (features), medium (enterprise pricing not listed)

## What it is
Enterprise workflow orchestration platform built on OSS Conductor (originally Netflix), now with first-class agentic capabilities. Durable-execution/workflow engine; managed SaaS or customer-hosted. Raised $60M Series B (Apr 2026) for AI workflow orchestration.

## Hosted run control plane: YES (strong)
- Mature workflow UI; "Every run is traceable end to end." Developer Playground: "a streamlined workflow UI, workflow template explorer." Genuine start/inspect/resume control plane.

## HITL / approval gates: YES (explicit, productized)
- "Built in approvals, decision review, and escalation, plus full audit trails and the ability to resume workflows after human input." Human tasks are a documented primitive.

## Durable state & provenance: YES (core)
- "Durable execution for real operations: retries, timeouts, compensation, scheduling, event driven triggers, and deep observability." Full audit trails.

## Multi-agent: YES
- "Coordinate multiple specialized agents in one workflow, with ordering, parallelism, retries, and timeouts handled explicitly." LLM tasks, vector DB tasks, prompt mgmt w/ RBAC.

## Buyer / deploy
- Buyer: enterprise/platform teams; framing targets the "Agent Execution Gap" ("Agents shine in POCs but fail in production").
- Deploy: OSS (conductor-oss.org) + Orkes Hosted (SaaS) + Customer Hosted (customer cloud/on-prem, dedicated VPCs).

## PRICING (orkes.io/pricing)
- Developer Edition: "Free" — sandbox for prototyping, no SLA, not for production.
- Enterprise: "Custom" — "Up to 99.99% availability SLA," "Hosted on Customer cloud accounts or on-prem."
- Numeric enterprise pricing: not publicly listed (contact sales).
