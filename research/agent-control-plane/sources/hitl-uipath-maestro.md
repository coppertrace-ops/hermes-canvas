## What it is
Orchestration layer coordinating UiPath agents + third-party LLM agents (Claude, OpenAI, Gemini, Copilot) + robots + APIs + people under one governance/audit/observability layer. HITL exceptions routed to humans via Action Center.

## Verbatim (per site/docs, via search)
- "Maestro routes exceptions to humans via Action Center, supports pause, resume, retry, rewind, and skip on live process instances, and captures every step in an audit trail."
- "keeps humans in control where judgment is needed and wraps AI and automation in deterministic guardrails."
- "detailed audit trails, version control, and role-based access... regulated environments like finance, insurance, and healthcare."

## HITL model
**Exception-routed human tasks + full audit trail + live-instance control (pause/resume/rewind/skip)**. Process-orchestration lineage (RPA). Strong provenance (audit trail) + control primitives that echo Hermes (pause/resume/stop). Enterprise process tool, not developer agent SDK.
