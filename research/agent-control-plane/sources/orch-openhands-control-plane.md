## What it is
Operational layer to manage the sprawl of AI *coding* agents deployed across an enterprise. Built on top of the OpenHands open-source software-engineering agent. Positioned as the "software agent control plane."

## Verbatim quotes
- "It provides centralized control over how agents operate across repositories, teams, and environments"
- "It enforces access policies, tracks usage and cost, and provides a complete record of agent activity."
- "Access is scoped, activity is auditable, and execution is contained within defined boundaries."
- "Every conversation is logged and tied to a user. You can trace what happened, when it happened, and why."
- "Usage is tracked at the organization, user, and conversation level."
- "OpenHands Enterprise provides a fully self-hosted, on-premise system for running AI agents across an organization."

## Relevance to Hermes
Closest branded "agent control plane" in the coding-agent space. Emphasis on audit record, per-conversation tracing, cost attribution, sandboxed execution. DevOps/governance-first, self-hosted enterprise. Overlaps Hermes on "complete record of agent activity" + provenance, but framed for security/cost governance of a fleet, not human-first legibility of a single run.
