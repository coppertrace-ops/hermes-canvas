## What it is
Agentforce 3's Command Center = "a complete observability solution... a unified pane of glass to monitor agent health, measure performance, and optimize outcomes." Positioned to solve "the biggest blockers to scaling AI agents: visibility and control." OpenTelemetry-based tracing + Data Cloud.

## Verbatim (per site/press, via search)
- "Command Center rolls up all your agent activity, metrics and telemetry into a single unified dash."
- "deep insight into agent performance, with detailed analytics for error rates, escalation frequency, agent latency."
- "managers the ability to set real-time alerts in case an agent isn't behaving as expected."
- "Built-in session tracing via OpenTelemetry and Data Cloud."
