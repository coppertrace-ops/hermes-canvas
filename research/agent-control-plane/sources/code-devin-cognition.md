- Enterprise: custom — SAML/OIDC SSO, centralized controls, VPC deployment option.
- Concurrent sessions: Free/Pro "Up to 10"; Max/Team "Unlimited".
- Models: "OpenAI, Claude, and Gemini frontier models" from Pro up.
- NOTE: current pricing page is quota/seat-based; ACU pricing (below) is the older Devin 2.0 model.
