## Sierra
CX agent platform. "Experience Manager is a platform for observability, evaluation, reporting, and optimization of AI agents." Supervisory agents + deterministic guardrails: "Every production agent runs with supervisory agents watching for ambiguous or sensitive situations." Provenance: "understand the reasoning behind every agent action... view knowledge sources, systems accessed."
- HITL model: **AI-supervisor + guardrails + monitor/report** (mostly agent-supervises-agent; humans review analytics/escalations), not human per-action approval gates.
- Buyer: enterprise CX leaders. Pricing: outcome-based, not public.

## Decagon
AI customer-support concierge. **Watchtower** = "always-on QA system that reviews every conversation (AI and human) against custom scoring rubrics." **Trace View** = "trace every AI decision, see which model was called, which workflow triggered, which knowledge article was referenced." Escalation to humans "with full context."
- HITL model: **review-after QA + escalation + trace/provenance**; supervisor layer "catches errors before customers see them."
- Buyer: enterprise CX/support. Pricing: not public.

## Relevance
Strong on provenance/evidence + monitoring, but **domain-locked to customer support**, business-buyer, closed platforms. Not developer-facing general agent supervision.
